# records

* käyttäjä voi luoda tunnuksen ja kirjautua sisään
* käyttäjä voi lisätä albumeita sovellukseen
* käyttäjä voi antaa albumeille arvosanan (1-10)
* käyttäjä voi kirjoittaa arvostelun albumista
* käyttäjä voi nähdä muiden käyttäjien arvosteluja
* käyttäjä voi tarkastella yksittäisen albumin tietoja ja arvosteluja
